# XploraKōdo Frontend

React + TypeScript + TailwindCSS frontend for the XploraKōdo coaching platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Wallet Dashboard with balance cards and transaction history
- Voice Coaching sessions
- Video Coaching sessions
- Assessment results

## API Configuration

Update `src/services/coachingApi.ts` to point to your backend API URL.
